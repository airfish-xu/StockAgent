# 牛散投资分析总控智能体系统

## 项目概述

这是一个基于多智能体架构的牛散投资分析系统，能够根据输入的牛散姓名，自动分析其持股情况，并提供买入/卖出时机的决策建议。

## 系统架构

### 核心组件

1. **总控智能体 (Master Agent)** - 协调所有智能体的工作
2. **数据收集智能体 (CollectInfo Agent)** - 收集牛散持仓数据
3. **辩论智能体 (Debater Agent)** - 进行多空观点辩论分析
4. **技术分析智能体 (Technical Agent)** - 分析股票技术指标
5. **决策智能体 (Decision Agent)** - 综合所有信息给出投资建议

### 数据流

```
输入牛散姓名 → 数据收集 → 技术分析 → 多空辩论 → 综合决策 → 投资建议
```

## 功能特性

- ✅ **图形用户界面** - 直观的Tkinter界面
- ✅ **实时分析过程显示** - 展示每个分析步骤的进度和结果
- ✅ **多智能体协调** - 自动调用四个专业智能体
- ✅ **模拟数据支持** - 在没有真实数据时提供演示数据
- ✅ **数据库管理** - SQLite存储持仓和分析结果
- ✅ **命令行和GUI双模式** - 灵活的使用方式

## 快速开始

## 使用说明

### 图形界面操作

1. 启动程序后选择"图形界面模式"
2. 在界面中输入牛散姓名（如：葛卫东）
3. 点击"开始分析"按钮
4. 观察分析过程，查看每个步骤的结果
5. 查看最终的投资建议

### 分析流程

1. **数据收集** - 获取牛散的持仓数据
2. **技术分析** - 分析每只股票的技术指标
3. **多空辩论** - 进行多头和空头的观点辩论
4. **综合决策** - 基于所有分析给出投资建议

## 技术栈

- **编程语言**: Python 3.8+
- **GUI框架**: Tkinter
- **数据库**: SQLite
- **多进程**: subprocess模块
- **配置文件**: JSON格式

## 开发说明

### 添加新的智能体

1. 在对应的智能体目录中实现功能
2. 在`master_agent/config.py`中配置路径
3. 在`master_agent/agent_coordinator.py`中添加调用方法
4. 在GUI中集成显示逻辑

### 扩展数据源

修改`master_agent/data_manager.py`中的数据库操作，支持更多数据源。

## 故障排除

### 常见问题

1. **GUI启动失败**: 检查Python环境和Tkinter支持
2. **智能体调用失败**: 确认各智能体路径配置正确
3. **数据库错误**: 检查SQLite文件权限和完整性

### 日志查看

所有分析过程都会在控制台输出详细日志，便于调试。
